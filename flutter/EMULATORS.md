# 模拟器配置建议

建议在 Android Studio 中创建 **3 类尺寸** 的模拟器，覆盖绝大多数安卓手机：

## 1. 小屏手机

- **机型示例**：Pixel 4a、Galaxy S10e
- **分辨率**：约 1080×2340（~5.8 英寸）
- **逻辑分辨率**：约 360×780 dp
- **用途**：验证小屏下的布局、滚动、按钮可点性

## 2. 普通主流手机

- **机型示例**：Pixel 6、Galaxy S21
- **分辨率**：约 1080×2400（~6.2 英寸）
- **逻辑分辨率**：约 412×915 dp
- **用途**：主要测试机型，代表大多数用户

## 3. 大屏手机

- **机型示例**：Pixel 7 Pro、Galaxy S23 Ultra
- **分辨率**：约 1440×3120（~6.7 英寸）
- **逻辑分辨率**：约 412×915 dp 或更大
- **用途**：验证大屏下的留白、图表展示

## 设计规范（已应用）

- 边距：16 dp
- 卡片圆角：12 dp
- 按钮高度：48 dp
- 图表宽度：占满父容器
- 页面：支持上下滚动
